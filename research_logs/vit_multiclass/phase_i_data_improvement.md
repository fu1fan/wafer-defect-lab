# ViT Phase I: Data-Level Imbalance Ablations (CAFormer-S18)

## 目标

在当前最优 ViT 模型 (CAFormer-S18 C1, score=0.8054) 基础上，复现并对比 ResNet50 Phase I 的数据层改进实验，验证：
1. 数据层干预对 ViT 的效果是否与 CNN 不同
2. 是否能发现可提升 ViT 的数据处理方法

---

## 实验配置

**基线模型**: CAFormer-S18 (caformer_s18_wafer)  
**基线配置**: configs/modal/research_vit/experiments/C1_caformer_s18_lowlr_warmup.yaml  
- lr=1e-4, cosine_warmup (2ep), focal γ=1.5, CB sampler, wd=1e-3

**评分函数**:
```
score = 0.40 × macro_f1 + 0.25 × macro_recall + 0.20 × minority_recall_mean + 0.15 × accuracy
minority = mean(Center, Donut, Loc, Near-full, Scratch)
```

所有 Phase I 实验均使用 **8 epoch** 以节省时间（基线验证用相同配置）。

## 关键发现

### 1. I5 (α=0.5 平滑重采样) 是明显赢家 (+0.020)
- 比基线高出 0.020，是迄今为止数据层干预中最大的单项提升
- 更激进的重采样 (α=0.5) 比温和的 α=0.75 效果好很多（0.8057 vs 0.7883）
- mF1 从 0.7513 → 0.7755，Accuracy 从 0.9538 → 0.9683

### 2. ViT vs ResNet50 对数据增强的响应截然不同

| 实验 | ResNet50 结果 | ViT 结果 |
|------|--------------|---------|
| I2 类感知增强 | **灾难性**：−0.266，score=0.5221 | 温和正面：+0.006 |
| I1 平滑重采样 α=0.75 | 负面：−0.032 | 轻微正面：+0.002 |
| I3 联合 | 负面：−0.027 | 负面：−0.012 |
| I5 平滑重采样 α=0.50 | 负面 (同类型) | **显著正面：+0.020** |

**结论**：ViT 的注意力机制对数据分布变化和增强策略更鲁棒，ResNet50 对激进的类感知增强极度敏感。

### 3. 过度组合会互相干扰
I3 (I1+I2 combined) 效果差于 I1 和 I2 单独使用，说明数据层干预叠加存在负向交互效应。

### 4. 随机擦除 (I4) 对 ViT 训练不稳定
I4 val_acc 在训练中出现大幅波动 (0.9607 → 0.8803 → 0.9423)，说明擦除会破坏空间注意力学习。
